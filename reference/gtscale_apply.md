# Apply only the color component of a `gtscales` spec

Apply only the color component of a `gtscales` spec

## Usage

``` r
gtscale_apply(data, spec)
```

## Arguments

- data:

  A `gt_tbl` created by
  [`gt::gt()`](https://gt.rstudio.com/reference/gt.html).

- spec:

  A `gtscale_spec`.

## Value

A modified [gt::gt](https://gt.rstudio.com/reference/gt.html) table.
